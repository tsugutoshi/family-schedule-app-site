FamSync（家族カレンダー＆共有）

FamSync は、家族の予定や写真、掲示板のやり取りをひとつにまとめる家族向けアプリです。 AIの要約やイベント抽出で、面倒な整理もすっきり。 Flutter × Firebase によるクロスプラットフォーム（iOS / Android）対応です。

主な機能

  • 家族カレンダー：家族の予定・行事を色分け表示。誕生日・祝日なども見やすく整理。
  • 写真共有：高画質写真を家族で共有（HEIC対応・自動サムネイル生成）。
  • 掲示板・コメント：日々の連絡や相談をスレッド形式で管理。
  • タスク・チェックリスト：買い物や支度の抜け漏れ防止に。期限超過は赤強調表示。
  • 通知：作成・更新・リマインドのプッシュ通知（FCM）。
  • AI活用（無料範囲）：掲示板コメントの要点整理、テキストからイベント抽出など。

プランについて

  • ベーシックプラン（無料）：家族人数上限 3 名。基本的な利用が可能。
  • ファミリープラン（月額 480 円）：家族人数上限 5 名。家族での共有・AI分析の主要機能が利用可能。
  • プレミアムプラン（準備中）：現在は購入できません。

技術スタック

  • Flutter（Material 3）
  • Firebase（Auth / Firestore / Cloud Functions / Messaging / Storage）
  • Google Gemini API（AI要約・イベント抽出）
  • Stripe / Google Play / App Store 決済対応
家族の予定も写真も連絡も、ひとつにまとめよう。

🥬 Rezo — 冷蔵庫の管理者

Rezo は、冷蔵庫の在庫管理・レシピ提案・買い物最適化を AI で実現する パーソナル・シェフアプリです。カメラで食材をスキャンするだけで在庫を登録し、 賞味期限の通知やレシピの自動提案でフードロスを削減します。

主な機能

  • AI 食材スキャン：カメラ撮影・バーコード読み取りで食材を自動認識・登録。
  • レシート読み取り：買い物後のレシートを撮影し、食材を一括登録。
  • 在庫管理：食材名・数量・カテゴリ・賞味期限・保管場所を一元管理。
  • AI レシピ提案：冷蔵庫の在庫をもとに最適なレシピを自動生成。栄養バランス（PFC）も表示。
  • 買い物リスト：レシピから不足食材を自動提案。手動追加も可能。
  • 賞味期限通知：期限が近い食材をプッシュ通知でお知らせし、フードロスを防止。
  • 音声入力：ハンズフリーで食材を登録。
  • 献立作成：数日分の食事計画を在庫ベースで立案。

AI の活用

  • Google Gemini API で食材画像を解析し、品名・カテゴリを自動判定。
  • 在庫状況に応じたレシピ生成と栄養バランスの算出。
  • ユーザーがオプトインで食材画像を提供 → 独自モデルの精度向上に活用。

技術スタック

  • Flutter（Material 3）/ Riverpod
  • Firebase（Auth / Firestore / Cloud Functions / Messaging / Storage / App Check）
  • Google Gemini API / ML Kit / Vertex AI（食材認識・レシピ生成）
  • RevenueCat（サブスクリプション）/ Google AdMob（広告）
冷蔵庫の食材を、ムダなくおいしく。
Rezo で毎日の料理をもっとスマートに。
