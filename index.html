Family Schedule App

Family Schedule App は、家族全員の予定・タスク・コミュニケーションをまとめて管理できる Flutter 製のクロスプラットフォームアプリです。AI がホワイトボードやディスカッションの内容を解析し、 実行可能なタスクやイベントを自動で抽出。参加メンバーの割り当て、リマインド、情報共有をスマートに支援します。

主な機能

  • ホワイトボード： 家族会議の議事録やアイデアをリアルタイム編集。自動保存・リンク検出で情報整理もスムーズ。
  • ディスカッション： コメントで意見交換。投稿内の URL は自動的にリンク化され、ワンタップで参照できます。
  • AI分析： Gemini API を利用してタスク・イベントを抽出。必ず具体的なタスクを生成し、必要に応じて家族全員を関係者に追加。
  • チェックリスト管理： テンプレート作成、AI 生成、担当者アサインなどで ToDo を柔軟に運用。
  • 通知と既読管理： Firebase Cloud Messaging によるプッシュ通知、既読状況の可視化で情報の取りこぼしを防止。
  • マルチプラットフォーム対応： Flutter × Firebase で iOS / Android に同時展開。Firestore／Functions／Auth 等を活用。

AIサポートの活用例

  • ホワイトボード＋ディスカッション内容から「決定事項」や「チェックリスト」をワンクリック生成。
  • タスク抽出ダイアログで期日・担当者・関連イベントを自動設定。抽出結果はクラウドに即反映。
  • AI とのチャットで議題整理やアクションプランの相談が可能。

技術スタック

  • Flutter（Material 3 対応 UI）
  • Firebase（Auth / Firestore / Cloud Functions / Cloud Messaging / Storage）
  • Google Gemini API（AI解析・チャット）
  • Stripe 連携や広告バナーなどのマネタイズ基盤
家族の予定もタスクも、もっとスムーズに。 今日から Family Schedule App で、みんなの「やること」と「やりたいこと」を一元管理しましょう。
