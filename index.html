FamSync（家族カレンダー＆共有）

FamSync は、家族の予定や写真、掲示板のやり取りをひとつにまとめる家族向けアプリです。 AIの要約やイベント抽出で、面倒な整理もすっきり。 Flutter × Firebase によるクロスプラットフォーム（iOS / Android）対応です。

主な機能

  • 家族カレンダー：家族の予定・行事を色分け表示。誕生日・祝日なども見やすく整理。
  • 写真共有：高画質写真を家族で共有（HEIC対応・自動サムネイル生成）。
  • 掲示板・コメント：日々の連絡や相談をスレッド形式で管理。
  • タスク・チェックリスト：買い物や支度の抜け漏れ防止に。期限超過は赤強調表示。
  • 通知：作成・更新・リマインドのプッシュ通知（FCM）。
  • AI活用（無料範囲）：掲示板コメントの要点整理、テキストからイベント抽出など。

プランについて

  • ベーシックプラン（無料）：家族人数上限 3 名。基本的な利用が可能。
  • ファミリープラン（月額 480 円）：家族人数上限 5 名。家族での共有・AI分析の主要機能が利用可能。
  • プレミアムプラン（準備中）：現在は購入できません。

技術スタック

  • Flutter（Material 3）
  • Firebase（Auth / Firestore / Cloud Functions / Messaging / Storage）
  • Google Gemini API（AI要約・イベント抽出）
  • Stripe / Google Play / App Store 決済対応
家族の予定も写真も連絡も、ひとつにまとめよう。
FamSync で、家族のスケジュール管理をもっとスマートに。

🥬 REZO Cooking Navi （冷蔵庫の預言者）

Rezo は、冷蔵庫の在庫管理・レシピ提案・買い物最適化を AI で実現する パーソナル・シェフアプリです。カメラで食材をスキャンするだけで在庫を登録し、 賞味期限の通知やレシピの自動提案でフードロスを削減します。

主な機能

  • AI 食材スキャン：カメラ撮影・バーコード読み取りで食材を自動認識・登録。
  • レシート読み取り：買い物後のレシートを撮影し、食材を一括登録。
  • 在庫管理：食材名・数量・カテゴリ・賞味期限・保管場所を一元管理。
  • AI レシピ提案：冷蔵庫の在庫をもとに最適なレシピを自動生成。栄養バランス（PFC）も表示。
  • 買い物リスト：レシピから不足食材を自動提案。手動追加も可能。
  • 賞味期限通知：期限が近い食材をプッシュ通知でお知らせし、フードロスを防止。
  • 音声入力：ハンズフリーで食材を登録。
  • 献立作成：数日分の食事計画を在庫ベースで立案。

AI の活用

  • Google Gemini API で食材画像を解析し、品名・カテゴリを自動判定。
  • 在庫状況に応じたレシピ生成と栄養バランスの算出。
  • ユーザーがオプトインで食材画像を提供 → 独自モデルの精度向上に活用。

技術スタック

  • Flutter（Material 3）/ Riverpod
  • Firebase（Auth / Firestore / Cloud Functions / Messaging / Storage / App Check）
  • Google Gemini API / ML Kit / Vertex AI（食材認識・レシピ生成）
  • RevenueCat（サブスクリプション）/ Google AdMob（広告）
冷蔵庫の食材を、ムダなくおいしく。
Rezo で毎日の料理をもっとスマートに。

👕 FitCheck AI - Smart Size

FitCheck AI は、オンラインショッピングでの服のサイズ選びをサポートする AIアプリです。手持ちの服を基準に、購入検討中の服のサイズ表を読み取り、 フィット感をわかりやすく比較できます。

主な機能

  • マイクローゼット管理：よく着る服を写真付きで登録し、サイズの基準をまとめて管理。
  • サイズ表の読み取り：ECサイトのサイズ表をスクリーンショットや写真から取り込み。
  • サイズ比較：肩幅、身幅、着丈、袖丈などを手持ちの服と部位ごとに比較。
  • AI自動採寸：服の写真と参照物をもとに、AIが寸法を自動で推定。
  • ARプレビュー：サイズ差から着用時のきつさ・ゆるさを視覚的に確認。
  • 多言語対応：日本語、英語、韓国語、中国語、ドイツ語、フランス語、スペイン語に対応。

AI の活用

  • Google Gemini を活用してサイズ表の構造を解析し、寸法情報を自動抽出。
  • 登録済みの服とのサイズ差分をもとに、フィット感を具体的にアドバイス。
  • AI採寸とAR表示を組み合わせ、購入前の着用イメージを補助。

技術スタック

  • Flutter（Material 3）/ Provider
  • Firebase（Remote Config / Analytics / Crashlytics）
  • Google Gemini API / Google ML Kit（OCR・サイズ解析）
  • Google AdMob / App Store / Google Play 配信対応
ネット通販のサイズ選びを、もっと確実に。

📚 ManabiFlow — YouTube × AI 学習

ManabiFlow は、YouTubeの教育動画をAIが自動で構造化し、 受動的な"視聴"を能動的な"学習体験"へと変える次世代の学習プラットフォームです。 動画を検索するかURLを貼るだけで、AIが学習ゴール・クイズ・実践課題を自動生成します。

主な機能

  • AI カリキュラム自動生成：YouTube動画をAIが分析し、学習ゴール・難易度・クイズ・実践課題を自動作成。
  • クイズ＆実践課題：選択式クイズで理解度チェック。実践課題でアウトプットの機会を提供。
  • AIチューター：つまずいた時にその場で疑問を解決。24時間対応の家庭教師。
  • 適応学習エンジン：SM-2アルゴリズムの間隔反復システムで弱点を自動検出し、最適なタイミングで復習を提案。
  • デイリーチャレンジ：毎日届くチャレンジで学習を習慣化。ストリーク＆グレースデイ機能付き。
  • 学習ノート：手動メモ・AI自動要約・ハイライト抽出。タイムスタンプ付きで動画にジャンプ可能。
  • スキルバッジ：ブロンズ→シルバー→ゴールド→プラチナの4段階。QRコード付きでLinkedIn連携対応。
  • アナリティクス：学習時間・クイズスコア・完了数・ジャンル分布をグラフで可視化。
  • タイムライン：学習成果を投稿し、いいねやコメントで仲間と学び合うソーシャル学習空間。

料金プラン

  • Free（無料）：月3カリキュラムまで作成可能。基本的な学習体験をお試し。
  • Premium（月額 980 円）：カリキュラム無制限、AIチューター月20回、適応学習、アナリティクス、広告非表示。
  • Pro（月額 1,980 円）：全機能無制限。AIチューター使い放題、プラチナバッジ解放、優先サポート。

技術スタック

  • Flutter（Material 3）/ Riverpod
  • Firebase（Auth / Firestore / Cloud Functions / Messaging / Storage / Crashlytics / App Check）
  • Google Gemini API（動画分析・クイズ生成・AIチューター）
  • RevenueCat（サブスクリプション）/ Google AdMob（広告）
